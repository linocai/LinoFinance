v1 vs v1.1 · 一眼看完差距

v1 · 当前稳定版

  • iOS 强制 Light Mode，Tab Bar 系统默认样式
  • 总览 KPI 用普通卡片，无 Sparkline / 图表
  • 报表用静态 ThinBar，没有 Apple Charts 体验
  • macOS 三栏 OK，但无 ⌘K、无 Menu Bar Extra
  • 没有 Widget / Live Activity / Lock Screen
  • 没有 App Intents / Siri / Spotlight
  • 没有隐私模糊 / Face ID 解锁
  • 没有月度 AI 财务故事
  • 没有账户对账面板（应有 vs 实有）
  • 没有离线草稿队列与冲突解决
  • 金额没有 Tabular Numbers，对齐不稳

v1.1 · 设计目标

  • Liquid Glass 全端 + Light / Dark / High Contrast 自适应
  • Hero 净资产 + Sparkline + Apple Charts 报表
  • iOS 浮动 Tab Bar + 中央 FAB 一句话记账
  • Dynamic Island + Live Activity 实时还款 / AI 计划
  • macOS ⌘K 命令面板 + Menu Bar Extra + 多窗口
  • Widget 锁屏 / 主屏 / StandBy / Watch 复杂功能
  • App Intents + Siri + Spotlight + Shortcuts
  • Privacy Blur + Face / Touch ID + 后台模糊
  • AI 月报（可编辑 / 导出 PDF / 改语气）
  • Account Reconciliation + 一键 AccountAdjustment
  • SwiftData 草稿队列 + 冲突解决 + 失败重试
  • 金额全 mono + tnum + 永远「原币 + CNY」
